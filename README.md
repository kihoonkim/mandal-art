# 만다라트
이 프로젝트는 vite 를 통해 생성되었습니다.
```
npm create vite mandalart
```

React + Typescript + Styled Component 로 개발되었습니다.

## 사용된 라이브러리
- canvas: tldraw
- state: recoil
- auth: google apis : google-api-helper.ts

## Documents
- firestore: https://firebase.google.com/docs/firestore
